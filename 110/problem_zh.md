## 题目描述

给定整数 $N$，试把阶乘 $N!$ 分解质因数，按照算术基本定理的形式输出分解结果中的 $p_i$ 和 $c_i$ 即可。

## 输入格式

一个整数 $N$。

## 输出格式

$N!$ 分解质因数后的结果，共若干行，每行一对 $p_i,c_i$，表示含有 $p^{ci}_i$ 项。按照 $p_i$ 从小到大的顺序输出。

## 数据范围

$3 \leq N \leq 10^6$

## 样例

```input1
5
```

```output1
2 3
3 1
5 1
```

## 说明

$5! = 120 = 2^3 \times 3 \times 5$
